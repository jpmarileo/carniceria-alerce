# 🥩 Carnicería Alerce

**Página web oficial de Carnicería Alerce** – Calidad desde 1998 en Santiago, Chile.

---

## ✨ Descripción

Sitio web moderno y profesional para **Carnicería Alerce**, una carnicería tradicional chilena especializada en cortes premium de vacuno, cerdo y ave.

La página incluye:
- Carrusel destacado con los mejores cortes
- Catálogo interactivo con precios actualizados
- Carrito de compras funcional
- Pedido directo por WhatsApp

- **HTML5 + Tailwind CSS** – Diseño moderno y responsive
- **JavaScript puro** – Carrito de compras dinámico
